## 6. 数据流架构

### 6.1 日志创建/编辑流程

```mermaid
graph LR
    subgraph Client["前端"]
        Form["QSO编辑表单"]
        Validate["字段验证"]
        Cache["缓存层<br/>IndexedDB"]
    end
    
    subgraph Server["后端"]
        Recv["接收请求"]
        Auth["认证校验"]
        Parse["数据解析"]
        Save["保存到DB"]
        Response["返回响应"]
    end
    
    subgraph Storage["存储"]
        DB["MySQL"]
    end
    
    Form -->|提交| Validate
    Validate -->|✓合法| Cache
    Cache -->|同步| Recv
    Recv -->|验证| Auth
    Auth -->|✓通过| Parse
    Parse -->|处理| Save
    Save -->|写入| DB
    DB -->|确认| Response
    Response -->|返回| Cache
    Cache -->|更新| Form
    
    style Client fill:#e1f5ff
    style Server fill:#fff3e0
    style Storage fill:#f3e5f5
```

## 7. 认证和授权架构

```mermaid
graph TB
    subgraph Auth["认证流程"]
        Login["用户登录<br/>username+password"]
        Verify["验证密码<br/>bcrypt"]
        GenToken["生成JWT令牌<br/>AccessToken + RefreshToken"]
    end
    
    subgraph Autho["授权流程"]
        Request["API请求<br/>带Authorization头"]
        Decode["解析JWT<br/>验证签名"]
        GetUser["获取用户信息"]
        CheckPerm["检查权限"]
    end
    
    subgraph Token["令牌管理"]
        AccessToken["AccessToken<br/>7天有效期"]
        RefreshToken["RefreshToken<br/>30天有效期"]
        Refresh["令牌刷新<br/>无需重新登录"]
    end
    
    Login -->|bcrypt比对| Verify
    Verify -->|✓成功| GenToken
    GenToken -->|颁发| AccessToken
    GenToken -->|颁发| RefreshToken
    Request -->|包含| AccessToken
    AccessToken -->|验证| Decode
    Decode -->|提取user_id| GetUser
    GetUser -->|查询权限| CheckPerm
    CheckPerm -->|✓允许| Request
    AccessToken -->|过期| Refresh
    Refresh -->|使用| RefreshToken
    RefreshToken -->|颁发新| AccessToken
    
    style Auth fill:#ffccbc
    style Autho fill:#ffe0b2
    style Token fill:#ffb74d
```
