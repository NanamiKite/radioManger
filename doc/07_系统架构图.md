### DX Cluster 连接流程

```mermaid
sequenceDiagram
    participant User as 用户
    participant View as DXClusterView
    participant API as /api/v1/dxcluster
    participant Manager as DXClusterManager
    participant Node as DX Cluster节点

    User->>View: 选择节点 + 点击连接
    View->>API: POST /connect {node_host, node_port}
    API->>API: _resolve_callsign() 从激活位置获取呼号
    API->>Manager: connect(node, callsign)
    Manager->>Node: asyncio.open_connection()
    Node-->>Manager: login prompt
    Manager->>Node: 发送呼号
    Manager->>Manager: 启动 _listen_loop 后台任务
    Manager-->>API: status
    API-->>View: {success: true, status: {...}}

    View->>API: WebSocket /ws?token=jwt
    API-->>View: 推送历史 spot (最多100条)

    loop 实时
        Node->>Manager: spot行文本
        Manager->>Manager: parse_spot() 解析
        Manager->>View: WebSocket推送 {type: "spot", data: {...}}
    end

    User->>View: 点击断开
    View->>API: POST /disconnect
    API->>Manager: disconnect()
    Manager->>Node: 关闭 socket
    Manager-->>View: 推送 {type: "disconnect"}
```

## 13. 回收站流程

```mermaid
sequenceDiagram
    participant User as 用户
    participant FE as 前端
    participant API as /api/v1/logs
    participant Service as DeletedLogService
    participant DB as 数据库

    Note over User,DB: 删除台站触发回收站
    User->>FE: 删除台站
    FE->>API: DELETE /stations/{id}
    API->>Service: delete_station_logs()
    Service->>DB: 查询台站所有日志
    loop 每条日志
        Service->>DB: 备份日志JSON到 deleted_logs
        Service->>DB: 标记日志 is_deleted=True
    end
    Service->>DB: 位置软删除 + 台站软删除
    DB-->>FE: 删除完成

    Note over User,DB: 查看回收站
    User->>FE: 打开回收站页面
    FE->>API: GET /recycle/list
    API->>Service: get_deleted_logs()
    Service->>DB: 查询未恢复且未过期的条目
    DB-->>FE: 回收站列表 + days_remaining

    Note over User,DB: 恢复日志
    User->>FE: 点击恢复
    FE->>API: POST /recycle/{id}/restore
    API->>Service: restore_log()
    Service->>Service: 从log_data JSON创建新QSO
    Service->>DB: 标记 is_restored=True
    Service->>DB: 自动清理过期条目
    DB-->>FE: 返回恢复的日志
```

### 15.5 服务器模式: Admin 管理

```
┌────────┐  /admin/users  ┌────────┐  get_current_admin  ┌──────────┐

│        │               │        │

│        │               │        │
│        │  ←────────────│        │  AuditService.log()

操作审计链路:
  登录/注册 ──→ audit_logs: LOGIN / REGISTER
  导入日志   ──→ audit_logs: IMPORT_LOGS
  删除日志   ──→ audit_logs: DELETE_LOGS
  修改密码   ──→ audit_logs: CHANGE_PASSWORD
  管理员操作 ──→ audit_logs: ADMIN_TOGGLE_USER / ADMIN_RESET_PASSWORD / ...
  注销账号   ──→ audit_logs: DELETE_ACCOUNT_REQUEST / CANCEL / CONFIRM
```
