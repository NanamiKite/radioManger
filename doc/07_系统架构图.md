# 系统架构图文档

## 1. 整体系统架构

```mermaid
graph TB
    User["用户"]

    subgraph Client["客户端层"]
        Web["Web 浏览器<br/>Vue3 + Element Plus"]
        Electron["Electron 桌面<br/>应用"]
        Mobile["移动端<br/>浏览器"]
    end

    subgraph Network["网络层"]
        HTTPS["HTTPS / HTTP"]
        DevProxy["Vite Proxy<br/>/api → :8000"]
    end

    subgraph Backend["后端层 FastAPI"]
        Auth["认证<br/>JWT / Bcrypt"]
        API["RESTful API<br/>9个路由模块"]
        Middleware["中间件<br/>日志/错误/限速"]
    end

    subgraph Services["业务逻辑层"]
        LogService["日志服务<br/>CRUD+导入导出"]
        StationService["台站服务<br/>呼号管理"]
        LocationService["位置服务<br/>全局激活"]
        StatsService["统计服务<br/>计算与聚合"]
        QRZService["呼号查询<br/>QRZ + 缓存"]
        SyncService["同步服务<br/>GitHub"]
    end

    subgraph Storage["存储层"]
        DB[("SQLite / MySQL<br/>9张数据表")]
        FileSystem["文件系统<br/>ADI文件"]
        Cache["内存缓存<br/>呼号/查询"]
    end

    subgraph External["外部服务"]
        QRZ["QRZ.com<br/>XML API"]
        GitHub["GitHub<br/>API"]
    end

    User -->|访问| Client
    Client -->|HTTP| Network
    Network -->|请求| API
    API -->|验证| Auth
    API -->|经过| Middleware
    API -->|调用| Services
    Services -->|读写| DB
    Services -->|查询| External

    style Client fill:#e1f5ff
    style Backend fill:#fff3e0
    style Services fill:#f1f8e9
    style Storage fill:#efe5dd
    style External fill:#c8e6c9
```

## 7. 位置激活流程

```mermaid
sequenceDiagram
    User->>Frontend: 点击激活位置B
    Frontend->>API: POST /locations/{id}/activate
    API->>Service: activate_location(id)
    Service->>DB: 取消所有位置 is_active=False
    Service->>DB: 设置目标位置 is_active=True
    DB-->>Service: 更新成功
    Service-->>API: 返回激活位置
    API-->>Frontend: 200 OK
    Frontend->>User: 更新树形界面显示
    Note over Frontend: 日志列表自动切换到<br/>激活位置的台站过滤
```

## 11. 时区同步流程

```mermaid
sequenceDiagram
    participant User as 用户
    participant Settings as 设置页面
    participant API as 后端API
    participant DB as 数据库

    User->>Settings: 选择时区 Asia/Shanghai
    Settings->>API: PATCH /auth/me {timezone: "Asia/Shanghai"}
    API->>DB: 更新 user.timezone
    DB-->>API: 更新成功
    API-->>Settings: 200 OK
    Settings->>Settings: 同步更新 authStore.user
    Note over Settings: 关闭设置页面
    User->>Dashboard: 打开仪表板
    Dashboard->>Dashboard: 从 authStore.user.timezone 读取时区
    Dashboard->>Dashboard: 实时时钟切换到 Asia/Shanghai
```
