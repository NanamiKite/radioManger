# RadioManager - 业余无线电日志管理工具

## 项目概述

RadioManager是一款基于Electron开发的跨平台多功能业余无线电日志管理工具。提供日志管理、导入导出、呼号查询等功能，支持多种存储方案（本地、GitHub、云服务器），具有良好的可扩展性。

## 项目特性

- ✅ **多平台支持**：Windows、macOS、Linux通过Electron实现
- ✅ **灵活存储方案**：支持本地存储、GitHub云备份、云服务器部署
- ✅ **完整日志管理**：增删改查、导入导出、统计分析
- ✅ **呼号查询**：集成QRZ.com查询功能
- ✅ **用户权限管理**：用户认证、权限分离、数据隐私保护
- ✅ **性能优化**：支持树莓派等低内存设备
- ✅ **Docker部署**：三种部署模式均支持Docker容器化

## 技术栈

### 前端
- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **桌面应用**：Electron
- **UI框架**：Element Plus / Ant Design Vue
- **状态管理**：Pinia
- **HTTP客户端**：Axios
- **路由**：Vue Router

### 后端
- **框架**：FastAPI (Python 3.9+)
- **ORM**：SQLAlchemy
- **数据库**：MySQL 8.0+
- **任务队列**：Celery（可选）
- **缓存**：Redis（可选）
- **环境变量**：python-dotenv
- **数据验证**：Pydantic
- **日志**：Python logging

### DevOps
- **容器化**：Docker
- **编排**：Docker Compose
- **版本控制**：Git + GitHub

## 部署模式

### 1. 纯本地部署
- 前端：Electron应用
- 数据存储：本地.adi文件或SQLite
- 后端：可选本地FastAPI服务
- 适用场景：个人使用、离线环境

### 2. 局域网部署
- 前端：Electron应用或Web应用
- 后端：Docker容器化FastAPI + MySQL
- 部署位置：树莓派或NAS等本地服务器
- 网络访问：局域网HTTP/HTTPS
- 适用场景：小型俱乐部、家庭网络

### 3. 云服务器部署
- 前端：Web应用 + 可选Electron封装
- 后端：Docker容器化FastAPI + 云数据库
- CDN：静态资源加速
- 域名SSL：HTTPS加密传输
- 适用场景：多用户、远程访问、生产环境

## 核心模块

### 日志管理模块
- 日志导入（支持.adi、.adif格式）
- 日志展示（表格视图、详情视图）
- 日志编辑（编辑、新增、删除）
- 日志导出（支持多种格式）
- 日志统计（DXCC、WAZ、日期等维度）

### 用户认证模块
- 用户注册（用户名、邮箱、密码）
- 用户登录（JWT令牌认证）
- 密码重置（邮件验证）
- 权限管理（用户、管理员）

### 台站信息模块
- 呼号管理（验证业余无线电编码规则）
- 梅登网格（4位或6位）
- 电台参数（型号、天线等）
- 个人设置（时区、语言等）

### 呼号查询模块
- QRZ.com集成查询
- 查询结果缓存
- 离线查询支持

### 数据同步模块
- GitHub集成（Pull/Push）
- 云服务器同步
- 版本控制
- 冲突解决

## 关键特性说明

### 1. 数据缓存机制
- 新通联数据临时存储在缓存文件中
- 应用关闭时自动写入按日期命名的日志文件
- 支持手动保存和自动保存

### 2. 日志管理策略
- 不直接修改原始文件
- 创建工作副本用于操作
- 历史日志自动清理或手动管理
- 删除的日志保存在dellog文件夹

### 3. 数据验证
- 呼号格式验证（业余无线电编码规则）
- 梅登网格格式验证（4位或6位）
- 日期时间格式验证

### 4. 性能优化
- 减少内存占用（适配树莓派）
- 数据库查询优化（索引、分页）
- 前端虚拟滚动（大量日志显示）
- 缓存策略（接口响应缓存）

## 安全设计

- **密码加密**：bcrypt加密存储
- **传输安全**：HTTPS/TLS加密
- **数据隐私**：用户日志数据只对该用户可见
- **环境变量**：所有敏感配置使用环境变量
- **API认证**：JWT令牌认证
- **CORS策略**：明确的跨域资源共享策略
