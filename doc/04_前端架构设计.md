# 前端架构设计说明文档

## 前端技术栈

- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **桌面应用**：Electron
- **UI框架**：Element Plus / Ant Design Vue
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios
- **数据验证**：Vee-validate
- **日期处理**：dayjs
- **表格组件**：Vue Table / Vuetify DataTable
- **图表库**：ECharts / Chart.js
- **国际化**：i18n

## 数据缓存机制

### 缓存策略
1. **内存缓存**：Pinia store
2. **本地存储**：LocalStorage（<5MB）
3. **IndexedDB**：大文件存储（日志缓存）
4. **服务端缓存**：HTTP缓存头

### 通联缓存文件流程
```
1. 用户新增/编辑通联
2. 数据保存到Pinia store
3. 定时保存到缓存文件（IndexedDB）
4. 应用关闭时：
   - 检查缓存中是否有未保存数据
   - 提示用户保存
   - 按日期生成日志文件
   - 上传到服务器（如配置）
5. 应用启动时：
   - 检查缓存文件
   - 恢复未保存的数据
```
