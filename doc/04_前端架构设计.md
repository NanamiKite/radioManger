# 前端架构设计说明文档

## 前端技术栈

- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **桌面应用**：Electron（可选）
- **UI框架**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios
- **国际化**：vue-i18n
- **日期处理**：date-fns

## 核心页面设计

### 1. 认证流程

#### 1.1 登录页 (LoginView.vue)
- 用户名/密码表单
- Element Plus 表单验证
- 登录后 JWT token 持久化到 localStorage
- 跳转到 Dashboard

#### 1.2 注册页 (RegisterView.vue)
- 用户名+邮箱+密码+确认密码
- 密码强度验证（至少8字符）
- 注册成功后跳转到登录页

### 2. 仪表板 (DashboardView.vue)
- 欢迎语 + 用户名
- **UTC+0 时钟** 和 **用户时区时钟**（实时秒级更新）
- 统计卡片：Total QSO / DXCC / WAZ / QSL Sent/Received / LOTW Confirmed / Total Distance / Last QSO
- 快捷操作按钮：查看日志、管理台站、查询呼号、查看分析

### 3. 日志管理 (LogsView.vue)

#### 3.1 日志列表
- **过滤栏**：呼号模糊搜索、波段选择、模式选择、**台站选择**
- 默认只展示**激活台站**的日志
- 表格：Date / De Call / Call Sign / Band / Mode / Freq / RST / QSL / Comment / Operations
- 分页支持

#### 3.2 新建日志对话框
- 显示当前激活台站
- 呼号、日期、波段、模式、QSL状态、备注
- 呼号自动转大写

#### 3.3 日志详情 (LogDetailView.vue)
- 完整日志信息展示
- 编辑对话框
- 删除确认

#### 3.4 导入对话框
- `.adi/.adif` 文件拖拽上传
- 导入结果展示（成功数/跳过数/错误详情）

#### 3.5 导出对话框
- **台站选择**（必选）
- 日期范围、波段过滤
- 文件名：`{日期}_{台站呼号}.adi`

### 4. 台站与位置管理 (StationsView.vue)

**树状结构UI**，使用 `el-tree` 组件：

```
📡 BA7ABC (2 locations)
  ├─ ● Home              [OL63] [IC-7300]  [Edit] [Delete]
  └─ ○ Field Day         [OL53] [FT-818]   [Edit] [Delete]
📡 BH8XYZ (1 location)
  └─ ○ Mobile            [OL73] [FT-991A]  [Edit] [Delete]
```

- 单击 `<el-radio>` 激活位置
- **New Location** 对话框：关联台站、名称、网格、电台型号、天线型号、高度、QTH
- 编辑/删除通过按钮触发

### 5. 呼号查询 (CallsignSearchView.vue)
- 输入呼号 → 调用后端 → 缓存优先
- 展示：姓名、国家、网格、坐标、许可证信息
- 缓存/实时标记
- 链接到QRZ.com

### 6. 快捷链接 (ShortcutsView.vue)
- 卡片布局展示业余无线电相关链接
- 内置6个默认链接（QRZ、ARRL、LOTW、EHAM、DXWatch、PSK Reporter）
- 点击卡片在新窗口打开

### 7. 统计分析 (AnalysisView.vue)
- 统计卡片：Total QSO / DXCC / QSL Sent / LOTW Confirmed
- QSL 详细统计表
- 活动概况

### 8. 设置 (SettingsView.vue)
- **账户**：用户名、邮箱、角色、时区、注册时间
- **偏好**：语言切换、时区选择，保存到后端（`PATCH /auth/me`）
- **安全**：修改密码
- **关于**：版本、数据库模式、API地址

## 状态管理 (Pinia Store)

### useAuthStore
- `user`, `token`, `isAuthenticated`
- `login()`, `register()`, `logout()` — 登出时清空所有状态并跳转 `/login`

### useLogsStore
- `logs`, `stations`, `activeStation`, `pagination`, `filters`
- `fetchLogs()`, `fetchStations()`, `createLog()`, `updateLog()`, `deleteLog()`
- `refreshActiveStation()` — 通过激活位置反向查找激活台站

### useStatsStore
- `statistics`, `isLoading`
- `fetchStats()`

## 路由结构

```
/login              → LoginView
/register           → RegisterView
/dashboard          → DashboardView    [需要认证]
/logs               → LogsView         [需要认证]
/logs/:id           → LogDetailView    [需要认证]
/stations           → StationsView     [需要认证]
/callsigns          → CallsignSearchView [需要认证]
/shortcuts          → ShortcutsView    [需要认证]
/analysis           → AnalysisView     [需要认证]
/settings           → SettingsView     [需要认证]
/404                → NotFoundView
```

路由守卫：未认证访问需要认证的页面时自动跳转 `/login`。

## 国际化 (i18n)

支持语言：中文 (zh-CN) + 英文 (en-US)

覆盖范围：导航、认证、仪表板、日志、台站、分析、设置、呼号查询、快捷链接、验证错误、通用操作
