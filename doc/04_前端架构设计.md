# 前端架构设计说明文档

## 前端技术栈

- **框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **桌面应用**：Electron（可选）
- **UI框架**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router
- **HTTP客户端**：Axios
- **国际化**：vue-i18n
- **日期处理**：date-fns

## 核心页面设计

### 1. 认证流程

#### 1.1 登录页 (LoginView.vue)
- 用户名/密码表单
- Element Plus 表单验证
- 登录后 JWT token 持久化到 localStorage
- 跳转到 Dashboard

#### 1.2 注册页 (RegisterView.vue)
- 用户名+邮箱+密码+确认密码
- 密码强度验证（至少8字符）
- 注册成功后跳转到登录页

### 2. 仪表板 (DashboardView.vue)
- 欢迎语 + 用户名
- **UTC+0 时钟** 和 **用户时区时钟**（实时秒级更新）
- 统计卡片（6张）：总QSO / 本月QSO / 本年QSO / 已确认QSO / DXCC实体 / 台站数量
- 左列：最近 10 条日志（点击跳详情）+ 比赛日历（6 个近期国际比赛）
- 右列：DXCC 进度条（已通联/已确认/340）+ Band 分布柱状图 + Mode 分布柱状图
- 波段按频率从低到高排序
- 快捷操作按钮（9 个）：日志、台站、呼号、分析、工具箱、DX Cluster、快捷链接、回收站、设置

### 3. 日志管理 (LogsView.vue)

#### 3.1 日志列表
- **过滤栏**：呼号模糊搜索、波段选择、模式选择、**台站选择**
- 默认只展示**激活台站**的日志
- 表格：Date / De Call / Call Sign / Band / Mode / Freq / RST / QSL / Comment / Operations
- 分页支持

#### 3.2 新建日志对话框
- 显示当前激活台站
- 呼号、日期、波段、模式、QSL状态、备注
- 呼号自动转大写

#### 3.3 日志详情 (LogDetailView.vue)
- 完整日志信息展示
- 编辑对话框
- 删除确认

#### 3.4 导入对话框
- `.adi/.adif` 文件拖拽上传
- 导入结果展示（成功数/跳过数/错误详情）

#### 3.5 导出对话框
- **台站选择**（可选，未选时导出所有台站）
- 日期范围、波段过滤
- 文件名：`{日期}_{台站呼号}.adi`（无台站时为 `{日期}_AllStations.adi`）

### 4. 台站与位置管理 (StationsView.vue)

**树状结构UI**，使用 `el-tree` 组件：

```
📡 BA7ABC (2 locations)
  ├─ ● Home              [OL63] [IC-7300]  [Edit] [Delete]
  └─ ○ Field Day         [OL53] [FT-818]   [Edit] [Delete]
📡 BH8XYZ (1 location)
  └─ ○ Mobile            [OL73] [FT-991A]  [Edit] [Delete]
```

- 单击 `<el-radio>` 激活位置
- **New Location** 对话框：关联台站、名称、网格、电台型号、天线型号、高度、QTH
- 编辑/删除通过按钮触发

### 5. 呼号查询 (CallsignSearchView.vue)
- 输入呼号 → 调用后端 → 缓存优先
- 展示：姓名、国家、网格、坐标、许可证信息
- 缓存/实时/离线标记（离线时使用 DXCC 前缀推断）
- 链接到QRZ.com

### 6. DX Cluster 实时 Spot (DXClusterView.vue)
- **节点选择**：下拉选择预设 DX Cluster 节点（VE7CC/K1TTT/W3LPL 等）
- **连接管理**：连接/断开按钮，显示连接状态和运行时长
- **实时 Spot 列表**：WebSocket 推送的 spot 实时刷新
  - 显示：通报者、被叫台、频率、波段、模式、注释、时间
  - DXCC 实体自动识别
- **历史 Spot**：连接后一次性推送最近 100 条历史 spot
- **断连通知**：cluster 断开时前端收到 `{type: "disconnect"}` 通知

### 7. 梅登网格地图 (MapView.vue)
- **Leaflet 地图**（OpenStreetMap 底图），`maxBounds` 限制在全球范围内
- **两种显示模式**：
  - **网格模式**：所有 4 位 Maidenhead 网格矩形叠加，有 QSO 的格子按数量渐变填色（蓝→绿→黄→红），无 QSO 的显示淡灰色边框
  - **放射线模式**：从激活台站到每个有 QSO 的格子中心画线，已确认=绿色实线，未确认=蓝色虚线
- **网格标签**：缩放 ≥5 时显示半透明网格编号，有 QSO 的显示 `格子(count)`，缩放/平移时动态刷新
- **灰线昼夜图层**：checkbox 开关，基于 suncalc 太阳位置计算，夜间区域深蓝半透明填充 + 金色分界线
- **缩放滑块**：左侧音量条风格竖向滑块（2-12 级），与地图缩放双向同步
- **右侧浮动工具栏**：模式切换 + 灰线开关 + 选中格子信息 + 查看日志按钮 + 图例
- **联动**：点击"查看日志"跳转到 LogsView 并自动按网格筛选

### 8. 工具箱 (ToolsView.vue)
- **Tab 导航**，6 个工具：
  - **频率/波长**：频率↔波长互算 + 业余波段速查表
  - **偶极天线**：半波偶极长度计算 + SVG 天线结构图 + 波段速查表 + 线材类型下拉
  - **GP 垂直天线**：1/4 波长振子 + 地网长度（λ/4/λ/2 可选）+ 地网数量 + SVG 示意图 + 波段速查表
  - **SWR**：阻抗→SWR/Γ/反射功率/回波损耗/失配损耗 + SWR 反算 + SVG 传输线图 + 参考表
  - **LC 滤波器**：Butterworth/Chebyshev 低通/高通设计（1-9 阶）+ SVG LC 梯形网络图
  - **通用工具**：dB 功率转换 + AWG 线规查询
- 所有公式标注权威来源（ARRL Handbook / ITU-R / ASTM B258）

### 9. 回收站 (RecycleBinView.vue)
- 展示被删除的日志列表（7 天保留期）
- 每条显示：呼号、QSO 日期、波段、模式、删除原因、剩余天数
- **恢复操作**：点击恢复按钮将日志还原到正常列表
- 分页支持

### 10. 快捷链接 (ShortcutsView.vue)
- 卡片布局展示业余无线电相关链接
- 内置6个默认链接（QRZ、ARRL、LOTW、EHAM、DXWatch、PSK Reporter）
- 点击卡片在新窗口打开

### 11. 统计分析 (AnalysisView.vue)
- 统计卡片：Total QSO / DXCC / QSL Sent / LOTW Confirmed
- QSL 详细统计表
- 活动概况

### 12. 设置 (SettingsView.vue)
- **账户**：用户名、邮箱、角色、时区、注册时间
- **偏好**：语言切换、时区选择，保存到后端（`PATCH /auth/me`）
- **安全**：修改密码
- **关于**：版本、数据库模式、API地址

## 状态管理 (Pinia Store)

### useAuthStore
- `user`, `token`, `isAuthenticated`
- `login()`, `register()`, `logout()` — 登出时清空所有状态并跳转 `/login`

### useLogsStore
- `logs`, `stations`, `activeStation`, `pagination`, `filters`
- `fetchLogs()`, `fetchStations()`, `createLog()`, `updateLog()`, `deleteLog()`
- `refreshActiveStation()` — 通过激活位置反向查找激活台站

### useStatsStore
- `statistics`, `isLoading`
- `fetchStats()`

### useDXClusterStore
- `spots`, `status`, `nodes`, `isConnected`
- `connect()`, `disconnect()`, `fetchSpots()`, `fetchStatus()`
- `initWebSocket()` — 建立 WebSocket 连接，监听实时 spot 推送
- `closeWebSocket()` — 关闭 WebSocket

## 路由结构

```
/login              → LoginView
/register           → RegisterView
/dashboard          → DashboardView    [需要认证]
/logs               → LogsView         [需要认证]
/logs/:id           → LogDetailView    [需要认证]
/stations           → StationsView     [需要认证]
/callsigns          → CallsignSearchView [需要认证]
/dxcluster          → DXClusterView    [需要认证]
/recycle            → RecycleBinView   [需要认证]
/shortcuts          → ShortcutsView    [需要认证]
/analysis           → AnalysisView     [需要认证]
/settings           → SettingsView     [需要认证]
/404                → NotFoundView
```

路由守卫：未认证访问需要认证的页面时自动跳转 `/login`。

## 国际化 (i18n)

支持语言：中文 (zh-CN) + 英文 (en-US)

覆盖范围：导航、认证、仪表板、日志、台站、分析、设置、呼号查询、快捷链接、验证错误、通用操作

npm run electron-build # 打包桌面应用
